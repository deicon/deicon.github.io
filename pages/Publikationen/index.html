Unsere Publikationen

Die Erfahrungen aus zahlreichen kommerziellen Projekten, in denen Open Source Software zum Einsatz kamen mündeten in bisher zwei Büchern, erschienen im Markt und Technik Verlag.

Ziel war ein möglichst breit angelegter Überblick über eine Reihe wichtiger Open Source Projekte und deren Verknüpfung zu komplexen, verteilten Anwendungen.

Java mit Open Source

Finden Sie Ihren Einstieg in die professionelle Softwareentwicklung mit Java mit unserem Buch “Java mit Open Source Werkzeugen”. Anhand einer konkreten Fallstudie werden namhafte Werkzeuge und Projekte vorgestellt und deren Verwendung präsentiert.

J2EE mit Struts und Co.

Die zweite Auflage unseres Open Source Buchs mit Schwerpunkt auf J2EE und Struts. “J2EE mit Struts und Co.”. Dargestellt ist die Verwendung von Standard Web Frameworks zur Erstellung unternehmensweiter Anwendungen.
